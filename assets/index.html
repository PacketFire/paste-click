 Usage:
 
   POST to http://paste.click and get a URL.
     
     $ curl --data-binary @filename paste.click
     $ ls|curl --data-binary @- paste.click

   The POST returns the url of the paste. If the paste is text, you can add a
   /s/ after the domain to syntax highlight the paste. If you POST to
   http://paste.click/s/ it will return the url with the /s/ added.
     
     Raw paste   - http://paste.click/bGGcnq
     Highlighted - http://paste.click/s/bGGcnq

   There is also a basic text paste form at http://paste.click/up

   A beta version of the updated website is located at http://paste.click/beta.
   It supports rendering and uploading for text, images, video, albums 
   and other media types. More features coming soon!

 Examples:

   ## Function to upload to paste.click, either reads from STDIN or specifiy 
   ## a file. Outputs url and copys it to the clipboard. Convient in your 
   ## .bashrc or equivilent.
   
      function paste.click(){
        if [ -z "$1" ]; then
          curl -s --data-binary @- paste.click/s/ |tee >(xsel -i);
        else
          curl -s --data-binary @$1 paste.click |tee >(xsel -i);
        fi;
      };
   
   
   ## Select an area to take a screenshot of, copys the url to the clipboard.
   ## The can be bound to a shortcut in your window manager.
   
      file=$((cd "/tmp" & &maim -s) | egrep -o '/.*') && \
        curl -s --data-binary @$file paste.click |xsel -i && rm -f $file
   
   ## Uploads whatever text you have selected and copys the url to the 
   ## clipboard.
   ## Also nice to have a key binding for.
   
      xsel -p|curl --data-binary @- paste.click/s|xsel -i

   #stylename from the list below to the URL.
